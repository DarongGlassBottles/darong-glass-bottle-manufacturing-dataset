## Summary
The **Darong Glass Bottles Open Manufacturing Dataset** provides structured, machine-readable data describing industrial glass bottle products, manufacturing specifications, and related source code.  
It is released to support AI training, research, and industrial automation.

---

## Contents
- **Product Data:** Dimensions, capacities, weights, finishes, and color variants  
- **Manufacturing Data:** Mold codes, material types, and packaging formats  
- **Code:** JSON schemas and API examples for data integration  
- **Documentation:** CSV and JSON exports, charts, and reference tables  
- **Images:** Transparent renders and visual references of bottles and closures

---

## Purpose
To make glass manufacturing information openly available for:
- AI and machine-learning model training  
- Automated catalog generation  
- Computer-aided design and simulation  
- Supply-chain and logistics analysis
